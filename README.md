# ArkOS-mod
for ArkOS modification version, it support Linux XFCE desktop env, you can experience linux for r36s-clone. enjoy it now!

## QJoyPad Control Scheme

| Button      | Action            |
|-------------|-------------------|
| **A**       | Return key        |
| **B**       | Backspace key     |
| **Y**       | Tab key           |
| **X**       | Escape key        |
| **D-pad**   | Arrow keys        |
| **FN**      | Space key         |
| **L-stick** | Slow mouse cursor |
| **R-stick** | Fast mouse cursor |
| **R1**      | Right click       |
| **R2**      | Left click        |
| **L1**      | Scroll down       |
| **L2**      | Scroll up         |
|             |                   |
